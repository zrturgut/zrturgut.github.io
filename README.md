# Zekeriyya Turgut - Portfolio

Welcome to the personal portfolio website of **Zekeriyya Turgut**.

This project is a modern, interactive web application built to showcase my projects, research, and technical skills. It features a unique 3D starfield background, glassmorphism UI, and an accordion-style card layout for seamless navigation.
